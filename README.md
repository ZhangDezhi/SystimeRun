
#系统时间加速工具

## 说明
将系统时间加快或者减慢(甚至倒退)

## 环境
Linux需要安装tk模块 

```bash
#RHEL
$ yum -y install tkinter  tcl-devel tk-devel  #python2
$ yum -y install python36-tkinter  tcl-devel tk-devel  #python36

#Windows

```

## 使用
```bash

python __main__.py

```

## 恢复时间

Windows下使用脚本 timeSync.bat
